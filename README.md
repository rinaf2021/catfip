# Catfip.net
Проект предназначен для помощи в борьбе с опасным заболеванием у кошек под названием FIP (инфекционный перитонит кошек).

Проект планируется в трех решениях:
- Веб версия - расположен по ссылке https://catfip.rinaf.ru
- приложение для Android
- PWA

Разработка будет осуществляться поэтапно и распространяться свободно для всех пользователей.
